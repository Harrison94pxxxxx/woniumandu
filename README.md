# woniumandu
蜗牛慢读/微信小程序

一款阅读类的微信小程序，开发的主要目的是熟悉小程序api和语法，顺便练习基础技能。API源自http://novel.juhe.im/ 提供的的追书神器API。欢迎感兴趣或者喜欢的朋友clone到本地查看实际运行效果，欢迎Issue、Fork、Star😄。
